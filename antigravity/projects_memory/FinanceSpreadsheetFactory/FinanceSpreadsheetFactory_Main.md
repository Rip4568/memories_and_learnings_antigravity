# Finance Spreadsheet Factory - Memory

## Visão Geral
Projeto focado na criação e venda de planilhas financeiras premium (Excel/CSV) e no desenvolvimento de um Web App gerador de modelos customizáveis.

## Nichos de Alta Demanda (Identificados em Março 2026)
1. **B2B / Freelancer:** Fluxo de Caixa MEI, Precificação de Serviços, Gestão de Estoque para E-commerce.
2. **Vida Pessoal (High Ticket):** Planejador de Casamento Financeiro, Calculadora de Independência Financeira (FIRE), Controle de Dívidas.
3. **Investimentos:** Controle de Carteira de Dividendos, Calculadora de IR para Cripto, Gestão de Aluguéis (Airbnb).

## O que define o "Premium"
- **Design Visual:** Dashboards limpos, cores modernas (paletas pastéis ou dark mode elegante), fontes sans-serif legíveis.
- **UX (User Experience):** Células de fórmula protegidas, menus suspensos (Data Validation), formatação condicional inteligente.
- **Automação:** Macros (opcional, priorizar fórmulas nativas para compatibilidade mobile/Google Sheets) e tabelas dinâmicas.

## Stack Técnica (Web App Sugerido)
- **Frontend:** React + TypeScript + TailwindCSS.
- **Geração de Excel:** `exceljs` ou `sheetjs`.
- **Gráficos:** `recharts` ou `chart.js` para visualização em tempo real antes do download.

## Estratégia de Monetização
- **Lojas:** Etsy (Estilo de Vida), Gumroad (Técnico/B2B), Hotmart/Kiwify (Mercado Brasileiro).
- **Modelo:** Pagamento único por template ou Assinatura para o Gerador Web.
