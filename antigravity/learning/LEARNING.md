## Como Adicionar um Novo Aprendizado
1. Crie um novo arquivo markdown bem descritivo na pasta `learning/` (ex: `my_new_insight.md`).
2. Utilize o template padrão dentro do novo arquivo:
   ```markdown
   # [Date] - [Topic/Category]
   
   **Situation/Bug**: 
   [Describe the context or the error encountered]
   
   **Solution/Insight**: 
   [Describe functionality, fix, or best practice found]
   
   **Reference**: 
   [Link to docs or relevant file]
   ```
3. Adicione o link absoluto para o novo arquivo na lista abaixo.
